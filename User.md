# User Roles and Contributions

This document defines the core user roles within our Gamified Learning Platform and outlines the expected contributions for each.

## 1. Learner / Student
Learners are the primary users of the platform, focusing on improving their software development skills.

### Contributions:
- Complete coding challenges and courses to earn XP.
- Participate in community discussions and peer reviews.
- Climb the leaderboard through active engagement and continuous learning.

## 2. Mentor
Mentors are experienced developers who guide Learners through their journey.

### Contributions:
- Create new coding challenges and course material.
- Provide feedback on Learner submissions.
- Host live coding sessions and Q&A.

## 3. Administrator
Administrators oversee the platform's operations and ensure a smooth experience for all users.

### Contributions:
- Manage user accounts, roles, and permissions.
- Maintain platform health and oversee content quality.
- Update global gamification mechanics (e.g., XP algorithms, badges).
